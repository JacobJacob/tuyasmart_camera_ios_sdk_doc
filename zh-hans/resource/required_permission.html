1.1. 需要的权限

Camera SDK 需要使用到相册访问权限和麦克风使用权限，请确保您的项目中添加了下面两个权限的使用说明，否则在录制/截图和对讲时，会造成 App 崩溃。

// 需要在info.plist 文件中添加以下权限
Privacy - Microphone Usage Description
Privacy - Photo Library Additions Usage Description
